== Description ==

Adds a verified badge to BuddyPress member profiles. Manually verify users, manage members, and upgrade to Pro for automations.

**Free Features**

* Verified badge displayed on BuddyPress member profiles
* Manually verify members by username or user ID
* View and manage all verified members in one place
* Nonce-secured admin actions
* No bloat — lightweight, no external dependencies

**Pro Features**

* Custom badge colour
* Upload your own badge icon
* Auto-verify by user role
* Auto-verify by account age
* Auto-verify by membership level
* Automation workflow builder

[Upgrade to Pro →](https://store.webbyjack.com/verify-for-buddypress-pro)

== Installation ==

1. Upload the `bp-verified-badge` folder to `/wp-content/plugins/`.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Navigate to **Verified Badge** in the WordPress admin menu.
4. Use the **Manual Verification** tab to start verifying members.

== Frequently Asked Questions ==

= Does this work without BuddyPress? =

No. This plugin requires BuddyPress to be installed and active.

= Can I change the badge colour? =

Badge colour customisation is available in the Pro version.

= Will my verified users be retained if I deactivate the plugin? =

Yes. Verification status is stored as standard WordPress user meta and is not removed on deactivation.
